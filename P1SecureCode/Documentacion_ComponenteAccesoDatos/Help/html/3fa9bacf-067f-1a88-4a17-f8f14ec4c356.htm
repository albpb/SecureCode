claseAccesoDatos Métodos

Informació tècnica sobre classes

El tipo claseAccesoDatos expone a los siguientes miembros.

Métodos
 NombreDescripción
Actualitzar Metodo para actulizar la base de datos, eliminar, añadir o actualizar la tabla de manera desconectada, atraves del DataSet
Connectar Metodo para conectar-se a la base de datos atraves de la cadena de conexio que se encuentra en el archivo de app.config, que a mes en el mateix metode encripta l'arxiu app.config.
Executa Metodo para actulizar la base de datos, eliminar, añadir o actualizar la tabla de manera directa (Conectados)
PortarPerConsulta(String) Metodo para traer la tabla que nos interese atraves de una consulta personalizda y guardar-la en un DataSet
PortarPerConsulta(String, String) Metodo sobrescrito de PortarPerConsulta en el que ahora ademas de recibir la consulta recibe el nombre de la Tabla
PortarTaula Metodo para traer la tabla que nos interese con todos sus datos de la base de datos y guardar-la en un DataSet
